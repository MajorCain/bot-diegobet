# Diego Bet Bot

Bot que analisa jogos e envia sugestões de apostas com base em estatísticas de gols dos times.

## Tecnologias
- Python
- API Football-Data.org
- Telegram Bot

## Execução local

Crie um arquivo `.env` ou defina as variáveis no ambiente:

- `TELEGRAM_TOKEN`
- `TELEGRAM_CHAT_ID`
- `API_KEY`
